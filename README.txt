USAGE:
1. create a folder named "Localisation" in MusicBee install dir. eg.  c:\program files\MusicBee\Localisation
2. put the .lng file into that folder
3. select this langauge in the preference.
4. restart MusicBee.

使用说明
1. 在MusicBee安装目录下创建名为"Localisation" 的文件夹
2. 将语言文件 (zh-CN.lng) 放入该目录
3. 在preference(首选项)处修改Language(语言)为zh-CN(简体中文)
4. 重启MusicBee.
